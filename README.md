# EG 

## Description
This a repository contains eg or "for example" files. That includes this README.  If a file is the service default, it will be prefixed with default_ . If the file is for a specialized configuration, it will be prefixed with the specific service configurations i.e nginx_postgresql_gitea.service. The idea is to have files that are easy to reference/download for various projects.
